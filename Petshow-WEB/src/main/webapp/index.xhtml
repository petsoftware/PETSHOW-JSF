Quem somos?

Dois amigos e colegas de trabalho se uniram para formar uma plataforma para ajudar a melhorar a relação entre pets e seus donos. Daí surgiu a plataforma PETSHOW que é uma combinação das tecnologias WEB e Mobile. Aqui ajudamos as ONG's, os donos e os PETSHOPS se relacionarem também entre si. Também é um abiemte para descontração e diversão, converse com seus colegas que também tem pets, troque conhecimento com outros criadores.
